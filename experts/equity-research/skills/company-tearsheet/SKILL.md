# 公司速览卡（Company Tearsheet）

## 功能说明

在 5 分钟内生成一份结构化的公司一页纸概览，供快速了解一家公司的基本面貌。适用于：
- 初次接触一家公司时快速建立认知
- 投资会议前的快速准备
- 批量筛选时的标的快照

## 工作流

### Phase 1：基础信息采集

1. 公司名称、代码、上市交易所
2. 行业分类（GICS/申万/Wind）
3. 成立日期、上市日期
4. 总部/主要市场
5. 员工人数
6. 管理层关键人物（CEO/CFO）

### Phase 2：业务描述

1. 一段话描述：这家公司做什么、怎么赚钱
2. 收入结构拆解（按业务线/产品/区域）
3. 核心竞争优势 / 护城河
4. 行业地位和主要竞争对手

### Phase 3：关键财务数据

1. 最新报告期的核心数据：
   - 收入、增速
   - 毛利率、净利率
   - ROE、ROIC
   - 资产负债率
   - 经营现金流
2. 过去 3-5 年趋势
3. 共识预测（如有）

### Phase 4：估值快照

1. 当前市值 / EV
2. 关键倍数：P/E、EV/EBITDA、P/B、P/S
3. 历史估值区间（过去 3-5 年 P/E Band）
4. 相对行业的溢价/折价

### Phase 5：股东与交易

1. 前十大股东
2. 机构/散户/内部人持股比例
3. 近期重要股东变动
4. 融券/做空数据（如有）
5. 日均成交额/流动性

### Phase 6：催化剂与风险

1. 未来 3 个月关键事件
2. 近期重要新闻/公告
3. Top 3 投资亮点
4. Top 3 风险因素

## 注意事项

- Tearsheet 追求的是"快速建立基本认知"，不是深度分析
- 优先展示最新的可获取数据，标注报告期
- 竞争对手列 3-5 家最直接的即可
- 催化剂只列确定性较高的硬日期事件
- 如果某项数据未能获取，标注 [未披露] 而非省略该栏
- 适合作为深度分析（initiating-coverage、earnings-analysis）的前置步骤
