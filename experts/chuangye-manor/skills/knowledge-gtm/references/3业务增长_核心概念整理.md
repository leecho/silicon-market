# 业务增长模块核心概念整理

> 来源：学习卡课程体系·文字稿·3模块业务增长

---

## 一、销售是一切的开始

### 核心观点
- **没有销售，没有业绩，没有业务进来，什么都是假的**
- 销售是企业要具备的**能力**，不是岗位
- 每一个人都是销售，每个人都要去尽可能提供给客户一些服务
- 没有客户，企业就没有存在的基础

### 销售认知误区
| 误区 | 正确认知 |
|------|----------|
| 销售只是销售部门的事 | 销售是企业能力，每个人都要具备 |
| 上了课就能成为好销售 | 销售是练出来的，只有去练，没有别的方法 |
| 客户只关心产品和价格 | 客户关心的是你能解决什么问题 |

---

## 二、L2C（Lead to Cash）—— 从线索到现金

### 客户心路历程（核心框架）

```
不知道你是谁 → 不讨厌你 → 好奇 → 相信 → 成交
     ↓              ↓          ↓        ↓        ↓
  潜在线索       线索      可能性    目标客户   成交客户
```

### 各阶段说明

| 阶段 | 客户心态 | 企业动作 |
|------|----------|----------|
| **潜在线索** | 完全不知道你是谁 | 找到目标人群，触达 |
| **线索** | 知道你是谁，但不讨厌 | 建立初步联系 |
| **可能性** | 对你好奇，想了解更多 | 持续沟通，传递价值 |
| **目标客户** | 对你有信心，愿意谈合作 | 进入销售阶段 |
| **成交** | 愿意买单 | 交付 + 继续影响复购 |

### 关键原则
1. **每个环节的启动能量不要太大** - 不要让客户跳过环节
2. **交朋友都不可能第一天就很熟** - 销售也是一样
3. **被动找上门的客户成为目标客户的机会比主动找的低很多**
4. **客户为什么买单？为什么继续买单？这是一整天都需要问的问题**

## 三、企业运营本质

### 最简单的企业运营图

```
企业 + 客户 + 满足需求 + 买单 + 产生利润 + 活下去
```

### 企业运营核心逻辑
1. **企业不能独立生存，必须要有客户**
2. 客户为什么买单？（不只是买产品，是买你能解决的问题）
3. 客户愿意复购吗？（持续价值）
4. 找到新客户，重复这个过程 → 企业发展

### L2C漏斗

```
潜在线索 → 线索 → 可能性 → 目标客户 → 创造项目 → 签订合同 → 交付
   ↓         ↓         ↓          ↓          ↓           ↓        ↓
  触达     建立联系    持续沟通    建立信心    谈合作      成交    交付+复购
```

## 四、销售的五个步骤与关键动作

| 步骤 | 关键动作 | 说明 |
|------|----------|------|
| 第一步 | 找到目标客户 | 识别谁是你的目标客户 |
| 第二步 | 建立联系 | 第一次接触，让对方不讨厌 |
| 第三步 | 了解需求 | 进入客户世界，理解他的问题 |
| 第四步 | 提供方案 | 针对需求提供解决方案 |
| 第五步 | 成交与交付 | 签合同 + 做好交付 + 影响复购 |

### 五步话术的关键
- **进入客户世界** - 不是卖产品，是帮客户解决问题
- 每次拜访都要问：客户的心路历程现在在哪个状态？

---

## 五、销售前的准备（拜访质量决定销售质量）

### 拜访前必须检查
1. **目标客户清晰吗？** - 知道要见谁，为什么见
2. **客户需求了解吗？** - 知道他的痛点是什么
3. **准备说什么？** - 有明确的拜访目标
4. **预期结果是什么？** - 这次拜访要达到什么

### Briefing 原则
- 做事前先 Briefing（说清楚）
- 事后要 Debrief（复盘）

## 六、建立信任关系

### 信任的核心
- **让对方感受到你的真诚**
- 信任不是技巧，是态度
- 1v1 沟通是建立信任最有效的方式

### 1v1 教练辅导的价值
- 破冰、建立悟性
- 一对一比一对多更有效
- 教练不是给答案，是引导对方找到答案

## 七、客户ABC分类

### 二维矩阵

| 维度 | 说明 |
|------|------|
| **价值** | 这个客户能给你带来多少收入/利润 |
| **潜力** | 这个客户未来能成长多少 |

### 分类结果

| 类型 | 价值 | 潜力 | 策略 |
|------|------|------|------|
| A类 | 高 | 高 | 重点投入资源 |
| B类 | 高 | 低 / 低 | 维护好关系，控制投入 |
| C类 | 低 | 高 | 培养潜力，控制成本 |

### 核心能力
- **建立客户的个人能力与企业能力** - 帮助客户成长，就是帮助自己成长

## 八、财务系统与业务增长的关系

### 财务预算是执行蓝图
- 预算决定企业将来的投资与行为
- 商业模型 → 量化 → 财务预算 → 行动计划

### 财务账套
- 从商业模型提取所有经济活动
- 成本和收入都要对应到具体活动
- CEO必须参与财务账套的设计

### 财务与业务的关系
```
商业模型 → 量化的商业模型 → 财务账套 → 财务预算 → 管理报表
     ↓              ↓            ↓          ↓          ↓
  企业策略      具体数字        经济活动    花钱计划    监督执行
```

## 九、销售三大基本功

| 基本功 | 说明 |
|--------|------|
| **领导力与影响力** | 真正的领导力是影响力，不是权力 |
| **沟通技巧** | 有效沟通是所有能力的基础 |
| **关系管理** | 建立信任，维护关系 |

### 沟通技巧核心
- **猴子管理** - 把猴子还回去，打造自驱型团队
- 不要替下属做决定，让他自己找到答案
- 有效开会 - 流程、角色、行为、记录、闭环

## 十、团队协作与组织

### 好的组织特点
- **扁平** - 让客户能快速接触你，让你能快速知道客户情况
- **不迷信组织模型** - 选对人比组织架构更重要

### 请人核心
- **价值观第一，能力第二**
- 没有正确价值观的人会把组织扭成官僚机构

### 四个阶段
| 阶段 | 核心 |
|------|------|
| **请** | 找对人，价值观第一 |
| **养** | 培养员工意识，给预算 |
| **用** | 沟通好就能用好 |
| **舍** | 果断，拖拖拉拉对双方都不利 |

## 补充创业庄园GTM主轴

### GTM核心要素（对照L2C）

| GTM要素 | 对应L2C环节 |
|---------|-------------|
| 目标人群 | 潜在线索 |
| 第一个钩子 | 线索（让客户不讨厌你） |
| 价值主张 | 可能性 → 目标客户（建立信心） |
| 关键活动 | 创造项目 → 签订合同 |
| 销售行动 | 成交 → 交付 → 复购 |

### GTM关键问题
1. 我的目标客户是谁？（不是在找客户，是在找**目标**客户）
2. 我的第一个钩子是什么？（让客户愿意听你说）
3. 客户的心路历程现在在哪一步？
4. 我能帮助客户解决什么问题？

---

## 总结：业务增长的三个关键

1. **先市场，后销售** - 没有市场洞察，销售是盲目的
2. **销售是一切的开始** - 没有业绩，其他都是假的
3. **企业运营是系统** - 商业模型、财务预算、执行管理要配套
