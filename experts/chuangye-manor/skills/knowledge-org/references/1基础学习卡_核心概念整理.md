## 一、什么是企业，为什么学企业

### 脑科学与情绪管理

#### 脑的基本架构
- **原始脑**（脑干部分）："打或跑"（Fight or Flight）的本能反应
- **大脑**（前额叶皮层）：理性思考、淡定处事
- 当大脑感受到压力 → 释放cortisol（皮质醇）→ 大脑"关机" → 原始脑接管 → 情绪化

#### 情绪化的本质
1. 对方批评你 → 大脑感到压力
2. 皮质醇上升 → 大脑关闭
3. 原始脑接管 → 声音提高、无法聆听
4. 唯一解决方法：**冷静，改变环境，让大脑有机会恢复**

#### 如何培养淡定心态
- **锻炼心态**：不轻易给大脑带来压力
- 批评可以听出正能量 → 大脑不感到压力
- 情绪不能压抑，要疏导
- 找让你有情绪的人来练，直到不舒服的感觉过去

#### 脑的可塑性
- 脑可以不停发展，老年人的脑也可以成长
- 脑的可塑性需要外部环境驱动（孟母三迁）
- **每次接到困难的工作就是塑造脑的机会，不能放过**

---

### 沟通与情绪层

#### 沟通的真相
- **沟通70%在情绪层，30%在内容层**
- 情绪层不和谐，内容无法传达
- 沟通的出发点如果是"说服"对方 → 对方抗拒 → 情绪层已经拒绝沟通

#### 语气的重要性
- 语气是连接情绪层的关键
- 开始语气没用好，对方一听就不爽，后面都是浪费时间

#### 沟通的四个步骤（沟通四部曲）
1. **扔出去** - 传递信息
2. **对方接到** - 确认接收
3. **对方懂** - 确认理解
4. **对方知道要做什么** - 产生行为

#### 有效的沟通
- 不能只追求"我说了"，要追求"对方产生了你要的行为"
- 很多沟通是失败的，因为最后一步没有发生
- 沟通前要考虑对方喜欢怎么接收信息

---

### 郁闷情绪的处理

#### 郁闷的正常过程
不接受 → 接受 → 郁闷 → 回到积极情绪

#### 解决方法
- **沟通和分享是郁闷的杀手**
- 跟人沟通，跟人分享
- 不要停在某个步骤里太久

## 三、学企业的三个境界

### 做人之道——赢取朋友与影响别人

> 核心书籍：《人性的弱点》（卡耐基，1936年出版）

#### 影响力的四个阶段

| 阶段 | 说明 |
|------|------|
| **1. 不要让人讨厌你** | 不批评、不抱怨、不责备 |
| **2. 让人喜欢你** | 真诚地关心别人、记住对方的名字 |
| **3. 让人信任你** | 真诚地倾听、让人感到被理解 |
| **4. 让人愿意受你影响** | 激发他人的主动性 |

#### 第一阶段：让人不讨厌你

**三不要：**
1. **不要批评** — 人永远不会承认自己错误，批评只会引发争吵
2. **不要抱怨** — 负能量影响所有人，包括你自己
3. **不要责备** — 责备没有用，要给方向

**真诚的表扬：**
- 人类最深层次的需求是被认同（Appreciation）
- 真诚的表扬能激发人的激情

---

### 六脉神剑——产生影响力的六个高效行为

| 行为 | 说明 |
|------|------|
| **1. 有效开会** | 有清晰目标、有输出标准、有角色分工 |
| **2. 一对一沟通** | 最重要的沟通方式，建立互信 |
| **3. 自我复盘** | 定期回顾自己的行为，找出问题 |
| **4. 表达能力** | 能够清晰表达自己的想法 |
| **5. 深层沟通** | 穿透情绪层，进入内容层 |
| **6. 时间管理** | 管理好猴子，不要让事情悬在空中 |

---

### 领导力与影响力

#### 领导力的定义
- **领导力 = 影响力**
- 职位带来的不是领导力，是权力
- 真正的领导力是让别人愿意跟随你

#### 能力方程式
```
能力 = 价值观 × 心态 × 沟通 × 知识
```

#### 价值观 vs 心态
- **价值观**：下意识，是习惯，主宰行为
- **心态**：上意识，是刻意的

#### 自私的价值观是职业经理人发展的最大障碍
- 只关注福利薪酬，不关注自己能提供什么价值
- 只希望别人配合，不考虑自己是否方便
- 将公司资源看成自己的资源

## 四、学习正确姿势

### 用框架来学企业

#### 知识不是力量
- 知识可能让你感觉良好
- 但知识要变成力量，需要**影响力**
- 影响力需要**行为改变**

#### 学习的目的是改变行为
- 如果学了半天行为一点没变，基本没学到什么
- 行为改变了，学习才算有效

#### 系统的补充
- 碎片式知识传播需要有系统的补充，否则容易断章取义

## 五、建立职场人脉

### 人脉的四个组成部分

| 组成部分 | 说明 |
|----------|------|
| **价值观** | 行为的基础 |
| **人脉** | 工作上需要协作的人 |
| **工作流程** | 沟通的机制 |
| **开放沟通** | 保持对外部的好奇 |

### 建立人脉的要点
- 不要等有事才找人
- 入职第一个礼拜"拜码头"
- 请教对方：有什么需要注意的？有什么可以帮忙的？
- 养成做事有交代的习惯

### 上司与下属的关系
- 是整个组织最关键的人脉关系
- 管理者必须主动建立安全的沟通环境

---

## 总结：学企业的三个境界

| 境界 | 核心 | 行为 |
|------|------|------|
| **初阶** | 知道什么是企业 | 理解商业模型、财务、人事组织 |
| **中阶** | 用框架学企业 | 用六个维度、落地流程来思考 |
| **高阶** | 做人做事一致 | 影响他人、建立团队、实现目标 |

---

> **林老师原话**："学习一切都是为了改变行为。如果你学了半天，行为一点都没有改变，你基本没有学到什么。"
