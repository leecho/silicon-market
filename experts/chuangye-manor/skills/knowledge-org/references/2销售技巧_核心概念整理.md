## 一、领导力与影响力

### 核心定义
- **领导力 = 影响力**
- 职位带来的不是领导力，是权力
- 真正的领导力是让别人愿意跟随你（粉丝效应）

### 能力方程式
```
能力 = 价值观 × 心态 × 沟通 × 知识
```

### 领导力的四个阶段（来自《人性的弱点》）
1. 不要让人讨厌你
2. 让人喜欢你
3. 让人信任你
4. 让人愿意受你影响

### 领导力修炼要点
- **招数不用多，能高效发挥简单的招数就够用**
- 招数要系统化，否则跟没学一样
- 招数不能被高效发挥 → 执行不到位 → 不是招数不灵
- 领导力是内功，不是权力

### 常见的负面行为（破坏影响力）
- 自私/以自我为中心
- 公私不分，将公司资源看成自己的资源
- 玻璃心/封闭心态（不敢尝试没把握的事）
- 只追求福利薪酬，不关注自己能提供什么价值

## 二、沟通技巧

### 沟通四部曲
1. **扔出去** — 传递信息
2. **对方接到** — 确认接收
3. **对方懂** — 确认理解
4. **对方知道要做什么** — 产生行为

### 沟通失败的常见情况
- 发邮件对方没看到（没考虑对方接收习惯）
- 以为对方懂了，但实际上没懂
- 没有确认最终行为

### 正确的沟通方式（以老张给小陈安排工作为例）
1. 发邮件说明任务
2. 打电话确认对方收到
3. 确认对方看懂要做什么
4. 对方承诺会做
5. 提出期望看到的行为（如：提早半小时到）

### 向上沟通要点（面试时就要问）
- 问：有没有一对一沟通的习惯？
- 如果没有，要求建立定期一对一
- 准备需要问的问题
- 紧张是正常的，不要装
- 说实话实说，不懂就说不懂
- 绝对不能说别人坏话
- 走之前确认下次时间

### 与上司一对一的流程
1. 确认时间限制
2. 让上司主导（除非上司没准备）
3. 说实话实说，不装
4. 不要说别人坏话
5. 走之前确认下次时间
6. 问期望值是什么，要做到什么程度

## 三、猴子管理

### 猴子的定义
- **猴子 = 下一步**（做一件事的下一步动作）
- 猴子不能死，要养着
- 事情没有下一步就停在那里

### 两个角色
| 角色 | 责任 |
|------|------|
| **执行者** | 负责找到解决方案，不将猴子交出去 |
| **监督者** | 监督执行，确保猴子被养好 |

### 负责任的执行者
- 遇到问题请教会老板，但不会把猴子交出去
- 找到可行方案才去问老板是否接受
- 积累自己的经历

### 不负责任的执行者
- 遇到问题马上找老板，将猴子交出去
- 自己不成长，白白浪费历练机会

### 监督者的角色
- 提供历练机会，不是替下属解决问题
- 如果每次都帮下属搞定，会培养出"遇到困难就找老板"的习惯
- 剥夺了员工成长的机会

## 四、Briefing（沟通）流程

### Briefing的核心
- 说清楚 → 对方接到 → 对方懂 → 产生行为

### 好的沟通案例（老张和小陈）
1. 发邮件说明任务
2. 打电话确认收到
3. 确认看懂任务内容
4. 明确期望行为（提早半小时到）
5. 小陈承诺做到

### 不好的沟通案例
- 发完邮件没确认
- 对方没收到都不知道
- 没有跟进，猴子死了

### Briefing的关键
- 考虑对方接收信息的习惯
- 主动跟进，确保沟通完成
- 作为启动沟通的人，要对最终结果负责

## 五、一对一沟通

### 为什么一对一重要
- 建立人脉、打破冰、建立互信
- 多人的会议很难深度沟通
- 部门之间有墙，需要一对一拆墙
- 同事之间不了解就不够信任

### 一对一的场景
- 刚入职与上司建立关系
- 跨部门协作前先破冰
- 深度反馈（多人场合不方便）
- 决定分道扬镳（不适合公开宣布）

### 一对一的准备
1. **设清晰目标** — 不是闲聊
2. **了解对方背景** — 可以向HR请教
3. **考虑对方期望值** — 不要只说自己的需求
4. **构思如何达成共识** — 求同存异

### 一对一的流程（适中控制）
1. 早到，在门口等
2. 约在小圆桌，不是大办公桌
3. 关手机，不接干扰电话
4. 先说时间限制（如30分钟）
5. 说今天沟通的目标
6. 充分聆听
7. 包容不同意见
8. 事后的跟进

### 包容 vs 盲目接受
- 包容：充分聆听、理解完整，再做判断
- 不是：对方说什么都接受

## 六、成为教练

### 三种角色的区别

| 角色 | 说明 |
|------|------|
| **老师** | 将知识教给你，你能否用是你的责任 |
| **顾问** | 分析情况，告诉你应该怎么做（不够靠谱） |
| **教练** | 挖掘你的潜力，让你知道自己不懂什么，然后去学 |

### 教练的核心
- 相信你自己能找到答案
- 让你知道问题在哪里，然后你自己找方法
- 不直接给答案，而是引导

### CEO的两大责任

#### 1. 营造高效团队协作环境
- 这个环境是拿来达到企业目标的

#### 2. 建立健康的沟通机制
- 确保每个员工都能得到良好发展
- 知道每个员工有不同的需求
- 上司与下属的关系是最关键的人脉关系

### 管理者必须是下属的教练
- 下属成功就是管理者的成功
- 培养员工是投资时间，不是花钱了事

## 七、建立信任关系

### 信任的关键词
- **真诚** — 让对方感受到你的真诚
- **主动** — 主动建立沟通环境
- **安全** — 让下属感到可以找你沟通

### 上司必须做的事
- 主动与下属建立沟通关系
- 建立安全的沟通环境
- 将下属的成功作为自己的KPI

### 基层管理者的重要性
- 基层管理者带着最前线的人打仗
- 如果基层管理者不知道什么是管理，执行力废了一大半

## 八、有效开会

### 开会前
- 设清晰目标
- 定输出标准（什么才算"开好了"）
- 设流程、时间、地点
- 订规矩（如：不带手机）

### 主持人（主持人心态）
- 开放包容
- 追求参与度
- 不批评任何人
- 聆听为主
- 控制时间、节奏

### 开会中
- 重复目标
- 说清楚输出标准
- 维持规矩
- 确保每个人参与
- 鼓励不同意见但不跑题

### 开会后
- 行为官反馈（观察大家的行为）
- 做记录（不是内容，是高效行为）

## 九、识别有效资源

### 融入新环境的四个方面

| 方面 | 说明 |
|------|------|
| **了解价值观** | 观察员工行为，了解企业文化 |
| **建立人脉** | 上司介绍 + 自己拜码头 |
| **熟悉流程** | 知道上下游是谁，怎么配合 |
| **开放沟通** | 对流程以外的事保持好奇 |

### 入职第一个月要做的
- 第一周：拜码头，认识所有需要协作的人
- 请教：有什么需要注意的？有什么可以帮忙的？
- 养成做事有交代的习惯

### 同事之间
- 不要等有事才找人
- 平常没事也可以约吃饭、喝咖啡
- 了解对方的工作，建立人脉
