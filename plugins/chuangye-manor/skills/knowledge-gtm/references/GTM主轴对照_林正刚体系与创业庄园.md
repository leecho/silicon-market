# GTM主轴对照：从林正刚体系到创业庄园

> 目的：建立两套体系之间的映射关系，让概念不混淆、不重复

---

## 一、核心概念对照表

### 林正刚体系 vs 创业庄园

| 维度 | 林正刚体系（学习卡/L2C） | 创业庄园（AGENTS.md） | 对照说明 |
|------|-------------------------|---------------------|----------|
| **切入市场** | GTM（Go-to-Market）= 从哪里切入市场 | GTM = 从哪里切入市场 | ✅ 定义一致 |
| **起点** | 潜在线索（不知道你是谁） | 目标人群（想走进去、但对方还没看见你） | 稍有差异：林体系从"不知道"出发，庄园从"想进去但没看见"出发 |
| **第一步** | 线索（知道你是谁，但不讨厌） | 第一个钩子（第一个让客户停下来的动作） | 两者角度不同：林体系描述客户心态，庄园描述企业动作 |
| **建立信心** | 可能性（好奇 → 相信） | 价值主张（你能解决什么人的什么问题） | 内涵相通，都是建立客户信任的过程 |
| **推进** | 目标客户（愿意谈合作） | 关键活动（推动客户往前走的关键动作） | 两者都属于"关键推进动作" |
| **成交** | 成交客户（愿意买单） | 销售行动（成交 → 交付 → 复购） | 一致，庄园更强调交付后的复购 |
| **完整链条** | L2C：潜在线索→线索→可能性→目标客户→成交 | 销售阶段：动作链条 | ✅ 框架一致 |

## 二、林正刚 L2C 详细映射

### L2C 完整漏斗

```
潜在线索 → 线索 → 可能性 → 目标客户 → 成交
   ↓         ↓         ↓          ↓        ↓
  触达     建立联系    持续沟通    建立信心    交付+复购
```

### 对应到创业庄园GTM

| L2C环节 | 创业庄园GTM要素 | 具体动作 |
|---------|----------------|----------|
| **潜在线索** | 目标人群 | 找到"想走进去、但对方还没看见你"的人群 |
| **线索** | 第一个钩子 | 让客户知道你是谁，并且不讨厌你 |
| **可能性** | 价值主张 | 建立客户相信：你能解决他的问题 |
| **目标客户** | 关键活动 | 推动客户从"考虑"到"愿意谈合作" |
| **成交** | 销售行动 | 签合同 + 交付 + 影响复购 |

## 三、关键概念澄清

### 1. "目标人群"的两种理解

| 体系 | 定义 | 含义 |
|------|------|------|
| **林正刚体系** | 潜在线索 = 完全不知道你是谁的人 | 强调"不知道"的阶段 |
| **创业庄园** | 目标人群 = 想走进去、但对方还没看见你 | 强调"有意愿但没发现"的状态 |

> **对照结论**：创业庄园的"目标人群"比林正刚体系的"潜在线索"更精准，因为它包含了"有意愿"的前提。

### 2. "第一个钩子"的核心

> 第一个让客户**停下来**的动作。

不是让客户立刻买单，而是：
- 让客户愿意多看你一眼
- 让客户知道你是谁 + 不讨厌你
- 为进入下一个阶段（可能性）做准备

### 3. "价值主张"的核心

> 你能解决什么人的什么问题。

不是：
- 你的产品是什么
- 你的公司有多好
- 你的价格有多便宜

而是：
- 客户的问题是什么
- 你的解决方案是什么
- 为什么客户要相信你

## 四、G2M思维框架（从林正刚到创业庄园）

### 四步对照

```
第一步：找对人（目标人群）
   ↓
   林正刚：潜在线索
   创业庄园：想走进去、但对方还没看见你
   
第二步：让对方注意到你（第一个钩子）
   ↓
   林正刚：线索（知道你是谁，但不讨厌）
   创业庄园：第一个让客户停下来的动作
   
第三步：让对方相信你（价值主张）
   ↓
   林正刚：可能性（好奇 → 相信）
   创业庄园：你能解决什么人的什么问题
   
第四步：推动成交（关键活动 + 销售行动）
   ↓
   林正刚：目标客户 → 成交
   创业庄园：关键活动 + 销售行动（成交 → 交付 → 复购）
```

## 五、G2M关键问题清单

### 自我诊断问题（林正刚体系视角）

| 问题 | 检查内容 |
|------|----------|
| 我的目标客户是谁？ | 不是"所有人"，是"想走进去但还没看见你"的人 |
| 我的第一个钩子是什么？ | 客户为什么会停下来看你？ |
| 客户现在在哪一步？ | 潜在线索/线索/可能性/目标客户/成交？ |
| 我现在在推哪个阶段？ | 不要跳过阶段 |
| 我能帮客户解决什么问题？ | 不是卖产品，是解决问题 |

### 常见错误对照

| 错误 | 正确 |
|------|------|
| "我的产品很好，客户会主动来找我" | 客户还不知道你，需要主动触达 |
| "我直接卖产品" | 先建立信任，再提供方案 |
| "客户应该会买单" | 客户在哪个心态阶段？每个阶段有对应的动作 |
| "成交就是结束" | 成交是开始，影响复购才是关键 |

## 六、创业庄园GTM主轴的补充维度

### 五个核心问题（来自学习卡）

```
1. 我的目标客户是谁？（不是"所有人"，是"目标"客户）
2. 我的第一个钩子是什么？（让客户愿意停下来听你说）
3. 客户的心路历程现在在哪一步？（L2C的哪个阶段）
4. 我能帮助客户解决什么问题？（不是卖产品，是解决问题）
5. 客户为什么买单？为什么继续买单？（这是创业第一天到每一天都要问的问题）
```

## 八、操作层面的GTM检查流程

### 每次见客户前，检查五件事：

```
□ 1. 我的目标客户清晰吗？（谁是我的目标客户？）
□ 2. 我准备说什么？（客户的问题是什么，我能解决什么）
□ 3. 客户现在在哪个阶段？（潜在线索/线索/可能性/目标客户）
□ 4. 我要推动客户往哪一步走？（这次拜访的目标）
□ 5. 拜访后客户应该做什么？（期望的下一个动作）
```

## 九、补充：销售五步 vs GTM主轴

| 销售五步 | GTM主轴 | 说明 |
|----------|---------|------|
| 找到目标客户 | 目标人群 | 第一步，确定谁是你的目标 |
| 建立联系 | 第一个钩子 | 让客户知道你，不讨厌你 |
| 了解需求 | 价值主张 | 不是卖产品，是理解客户的问题 |
| 提供方案 | 关键活动 | 针对需求提供解决方案 |
| 成交与交付 | 销售行动 | 成交是开始，交付和复购才是关键 |
