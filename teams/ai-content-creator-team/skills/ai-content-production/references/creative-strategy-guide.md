# 创意策略方法论指南

## 概述

本指南整合了系统化的创意策略方法论，为内容创作团队提供从商业意图到视觉执行的完整策略框架。适用于品牌定位、广告创意、场景设计、产品展示等多种创意场景。

---

## 一、创意探索框架

### 创意承诺量表

创意探索遵循"承诺递增"的路径设计——从轻量探索到重度生产：

```
轻量 ←─────────────────────────────────→ 重度
定位 → 情绪板 → 场景 → 卖点 → 广告 → 镜头 → 资产生产
```

| 阶段 | 用途 | 输出物 |
|------|------|--------|
| 定位(Positioning) | 明确"为谁做、做什么" | 定位文档 |
| 情绪板(Moodboard) | 确定视觉感受和方向 | 视觉方向描述 |
| 场景(Scene) | 产品在真实环境中的展示 | 场景方案 |
| 卖点(Offer) | 产品/服务的创意表达 | 卖点创意概念 |
| 广告(Ads) | 广告创意方向 | 广告创意简报 |
| 镜头(Shot) | 具体拍摄/生成参数 | 镜头表 |
| 标识(Logo) | 品牌视觉识别 | Logo概念方案 |

### 路径选择决策

```
用户意图分析：
├─ "我想推广我的产品" → 先定位，再广告方向
├─ "我想要一组品牌图" → 先情绪板，再场景
├─ "帮我设计个Logo" → 直接Logo方向
├─ "做一组社媒广告" → 先定位，再广告方向+镜头
├─ "拍产品图" → 先场景，再镜头
└─ "我不确定要什么" → 从定位开始，逐步推进
```

## 二、品牌/产品定位方法论

### 五要素定位模型

| 要素 | 核心问题 | 分析维度 |
|------|---------|---------|
| **受众** | 谁会看到？谁会购买？ | 年龄段、职业特征、消费能力、审美偏好、痛点需求 |
| **场景** | 何时何地接触到内容？ | 投放渠道、浏览设备、使用时段、消费心态 |
| **增长目标** | 这次要达成什么？ | 品牌认知/产品转化/用户留存/口碑传播 |
| **差异化** | 和竞品的核心区别？ | 功能差异/情感差异/价格差异/体验差异 |
| **排除项** | 绝对不能出现什么？ | 竞品元素/不当调性/敏感内容/风格禁区 |

### 定位输出模板

```markdown
## 品牌/产品定位

**产品**：[产品名称和简述]
**受众画像**：[核心受众描述，含人口统计和心理特征]
**使用场景**：[典型消费/使用场景]
**增长目标**：[本次创意的核心 KPI]
**价值主张**：[一句话差异化价值]
**排除清单**：
- [不能出现的元素1]
- [不能出现的元素2]
```

## 三、情绪板（Moodboard）方法论

### 情绪板构建五层

| 层 | 说明 | 产出 |
|----|------|------|
| 色彩层 | 主色、辅色、强调色及其情感联想 | 色彩方案（含色值） |
| 意象层 | 核心视觉意象和隐喻 | 3-5 个关键意象 |
| 质感层 | 材质、光泽、纹理特征 | 质感关键词 |
| 风格层 | 设计/摄影/艺术流派参考 | 风格定义 |
| 情绪层 | 3 个核心情绪关键词 | 情绪锚定 |

### 色彩情感映射

| 色系 | 情感联想 | 适用场景 |
|------|---------|---------|
| 暖白/米色 | 干净、信赖、日系 | 生活方式、极简品牌 |
| 深蓝/藏青 | 专业、稳重、科技 | B2B、金融、科技 |
| 珊瑚/粉橙 | 活力、年轻、女性 | 美妆、社交、生活 |
| 森绿/橄榄 | 自然、健康、可持续 | 食品、环保、户外 |
| 金/铜 | 高端、奢华、品质 | 奢侈品、高端服务 |
| 黑/深灰 | 力量、高级、神秘 | 时尚、汽车、高端科技 |

### 风格流派速查

| 风格 | 视觉特征 | 典型应用 |
|------|---------|---------|
| 极简主义 | 大量留白、单色/双色、几何线条 | 高端品牌、科技产品 |
| 新复古 | 颗粒感、怀旧色调、复古字体 | 咖啡/文创/独立品牌 |
| Y2K | 高饱和、金属光泽、渐变 | Z世代、潮流品牌 |
| 日式侘寂 | 不对称、自然纹理、灰调 | 生活美学、茶饮 |
| 孟菲斯 | 几何图形、撞色、波点条纹 | 活动/儿童/创意类 |
| 瑞士现代 | 网格系统、无衬线、信息层次 | 企业品牌、信息类 |
| 赛博朋克 | 霓虹色、暗背景、故障风 | 游戏、夜店、科幻 |

## 四、场景设计方法论

### 场景类型矩阵

| 场景类型 | 环境特征 | 适用产品 | 氛围要点 |
|---------|---------|---------|---------|
| 零售终端 | 货架/柜台/收银 | 快消品、零售 | 真实购物环境、产品突出 |
| 使用现场 | 用户实际使用环境 | 所有产品 | 自然、生活化、共鸣 |
| 体验空间 | 咨询/演示/交付 | 服务类 | 专业、舒适、信任 |
| 线上展示 | 落地页/详情页 | 电商 | 清晰、转化导向 |
| 社交传播 | 分享/打卡/互动 | 社媒营销 | UGC感、传播性 |
| 品牌空间 | 旗舰店/展厅 | 高端品牌 | 沉浸、品牌调性 |

### 场景描述公式

```
场景 = 环境（空间+光线+时间） + 产品（位置+呈现方式） + 人物（有无+动作+情绪） + 细节（道具+纹理+氛围）
```

### 场景光线参考

| 光线类型 | 效果 | 适用 |
|---------|------|------|
| 黄金时段 | 温暖、感性、电影感 | 生活方式、食品 |
| 硬朗自然光 | 清新、真实、日常 | 产品实拍、UGC |
| 影棚三点光 | 专业、立体、可控 | 产品摄影、电商 |
| 霓虹灯光 | 潮流、夜生活、科技 | 潮牌、游戏 |
| 柔和漫射光 | 柔美、梦幻、高级 | 美妆、婚纱 |

## 五、广告创意方向方法论

### 广告创意家族（Prompt Families）

每个广告方向可衍生为一个"创意家族"，包含：

1. **核心概念**：一句话创意主张
2. **视觉锚点**：核心视觉元素
3. **情感钩子**：引起目标受众情感共鸣的点
4. **行动指引**：CTA 方向
5. **变体延展**：3-5 个变体方向

### 渠道适配规则

| 渠道 | 尺寸 | 文字比例 | 核心约束 |
|------|------|---------|---------|
| 微信朋友圈 | 1:1 或 3:4 | ≤20% | 原生感、不像广告 |
| 抖音信息流 | 9:16 | 最少 | 3秒吸引、竖屏满屏 |
| 小红书 | 3:4 | 适中 | 种草感、真实感 |
| 淘宝主图 | 1:1 | 可高 | 卖点突出、白底优先 |
| B站 | 16:9 | 少 | 封面吸引力、信息量 |
| 户外大牌 | 宽幅 | 极少大字 | 远距离可读、5字内 |

### 20 变体批量策略

当需要系统性探索广告方向时，使用"20 变体矩阵"：

```
维度1：视觉风格（4种）× 维度2：内容角度（5种）= 20 个变体

视觉风格举例：
- A: 极简白底 + 产品特写
- B: 场景化生活方式
- C: 平面设计感（插画/图形）
- D: UGC 真实感

内容角度举例：
- 1: 功能卖点（解决什么问题）
- 2: 情感共鸣（用户故事）
- 3: 社会认同（数据/评价）
- 4: 稀缺紧迫（限时/限量）
- 5: 对比反差（前后/vs竞品）
```

## 六、镜头与构图方法论

### 景别体系

| 景别 | 画面范围 | 用途 | Prompt关键词 |
|------|---------|------|-------------|
| 远景 | 环境全貌 | 建立空间感 | wide shot, establishing shot |
| 全景 | 完整主体+环境 | 展示关系 | full shot |
| 中景 | 主体膝上 | 叙事互动 | medium shot |
| 近景 | 主体胸上 | 表情情绪 | close-up |
| 特写 | 局部细节 | 强调质感 | extreme close-up, macro |
| 微距 | 超近距离 | 材质/纹理 | macro photography |

### 视角体系

| 视角 | 心理效果 | 适用场景 |
|------|---------|---------|
| 平视 | 平等、客观、亲近 | 人物肖像、产品平视 |
| 俯拍 | 全局感、秩序感 | 美食摆盘、桌面布局 |
| 仰拍 | 力量感、崇高感 | 建筑、英雄感人物 |
| 鸟瞰 | 宏观、抽象、设计感 | 城市、场地布局 |
| 45度 | 立体感、自然感 | 产品展示（最常用） |

### 构图法则

| 法则 | 要点 | 适用 |
|------|------|------|
| 三分法 | 主体置于1/3线交叉点 | 通用 |
| 中心对称 | 主体居中、对称布局 | 产品正面、Logo |
| 前景框架 | 利用前景元素框住主体 | 场景感、纵深感 |
| 引导线 | 线条引导视线至主体 | 建筑、道路 |
| 负空间 | 大面积留白突出主体 | 极简设计、高端 |

## 七、Logo/标识概念方法论

### 标志类型选择

| 类型 | 特征 | 适用 | 示例参考 |
|------|------|------|---------|
| 字标(Wordmark) | 纯文字 | 名称独特的品牌 | Google, Coca-Cola |
| 字母标(Lettermark) | 首字母缩写 | 名称较长的品牌 | IBM, HBO |
| 图形标(Symbol) | 纯图形 | 全球化/高知名度 | Apple, Nike |
| 组合标(Combination) | 图形+文字 | 新品牌（最常用） | Adidas, Burger King |
| 徽标(Emblem) | 文字嵌入图形 | 传统/机构感 | Starbucks, Harley |

### Logo 方向描述公式

```
Logo方向 = 品牌名 + 标志类型 + 风格方向 + 字体类型 + 色彩方案 + 核心意象
```

## 八、发布级精修方法论（Generative Polish）

### 核心原则：精确层与生成层分离

在创作发布级视觉内容时，将工作分为两个独立层：

| 层 | 所有者 | 内容 |
|----|--------|------|
| **精确层**（确定性） | 文本工具/设计工具 | 文字、数据、Logo、尺寸、安全区域 |
| **生成层**（AI生成） | 图像生成工具 | 背景、纹理、光效、场景氛围 |

### 精修工作流

```
1. 定义资产体系
   └─ 目标格式、宽高比、渠道、审核状态

2. 锁定精确层
   └─ 创建文字/Logo/数据图表等确定性内容

3. 生成精修层
   └─ 用 AI 生成背景/纹理/光效（Prompt 中禁止文字/Logo/数据）

4. 确定性合成
   └─ 将精确层覆盖在生成层之上

5. 验证与交付
   └─ 检查尺寸、文字准确性、对比度、品牌一致性
```

### 精修 Prompt 规则

AI 生成精修层时，Prompt 必须包含以下约束：

```
创建一个高品质的视觉背景/精修层，用于[具体用途]。
禁止包含：文字、Logo、UI元素、图表、数字、产品声明、水印。
保留干净的负空间，用于后续叠加确定性内容。
使用高品质的光效、景深、纹理和构图，适合[目标渠道]。
```

### 精修适用场景

| 场景 | 精确层 | 生成层 |
|------|--------|--------|
| 社媒轮播图 | 标题/CTA/数据 | 背景纹理/场景 |
| 数据卡片 | 图表/数字/来源 | 背景氛围 |
| 产品落地页 | 产品截图/文案/按钮 | Hero背景/光效 |
| 发布海报 | 活动名称/日期/二维码 | 视觉主题/插画 |

### 护栏规则

- ❌ 不用 AI 生成最终的法律声明、价格信息、可用性声明
- ❌ 不依赖 AI 渲染的文字/数字/图表作为发布版本
- ❌ 不创建虚假产品 UI 或非授权品牌标志
- ❌ 不让视觉精修覆盖可读性、对比度和无障碍性

## 九、使用建议

### 对团队成员的指引

| 接收者 | 使用方式 |
|--------|---------|
| image-creator（珀西） | 接收创意策略后，按照 Prompt 建议生成图片 |
| video-generator（维欧） | 根据场景和镜头设计，生成对应视频 |
| copywriter（笔澜） | 参考定位和广告方向，撰写配套文案 |
| video-editor（柯立） | 根据镜头表和场景设计，进行后期处理 |

### 何时需要创意策划

| 信号 | 行动 |
|------|------|
| 用户说"不确定要什么风格" | 从定位开始 |
| 用户说"帮我设计一套品牌视觉" | 从情绪板开始 |
| 用户说"做一组广告" | 定位 → 广告方向 |
| 用户说"产品图怎么拍好看" | 场景 → 镜头设计 |
| 用户明确知道要什么 | 跳过策划，直接执行 |
