# 视频生成完整指南

## WorkBuddy 视频生成模型

本指南基于 WorkBuddy 平台内置的视频生成模型，所有生成操作通过 `多模态内容生成` Skill 自动触发。

---

## 一、可用模型

| 模型 | 调用参数 | 能力 | 特点 |
|------|----------|------|------|
| **HY-Video-1.5** | `hy-video-1.5` | 文生视频、图生视频 | 支持中英文，5-10秒高清视频，场景切换、多角色交互 |
| **YT-Video-2.0** | `yt-video-2.0` | 图生视频 | 动态连贯、画面过渡自然，适合广告/影视/产品展示 |
| **YT-Video-HumanActor** | `yt-video-humanactor` | 图生视频（人物） | 单张照片驱动人像视频，还原表情姿态，支持写实/二次元 |
| **YT-Video-FX** | `yt-video-fx` | 图片特效视频 | 上传图片+特效模板，静态图转动态特效视频 |

---

## 二、模型选择决策树

```
用户需求分析：
├─ 文生视频（无参考图）
│   └─ HY-Video-1.5（唯一支持文生视频的模型）
│       ⚠️ 质量优化：先生成关键帧图片，再转图生视频
│
├─ 图生视频（有参考图/关键帧）
│   ├─ 通用场景 → YT-Video-2.0（画面过渡最自然）
│   ├─ 需要中英文理解 → HY-Video-1.5
│   └─ 建议：YT-Video-2.0 优先
│
├─ 人物视频（人像照片驱动）
│   └─ YT-Video-HumanActor
│       ├─ 写实风格
│       └─ 二次元风格
│
└─ 特效视频（静态图+特效模板）
    └─ YT-Video-FX
        飘雪、粒子、光效等
```

## 三、核心策略：快速产出优先，按需升级

### 默认路径：直接生成

大多数场景下，应先尝试最短路径确保产出成功：
- **有参考图** → 直接图生视频（YT-Video-2.0）
- **无参考图** → 直接文生视频（HY-Video-1.5）

### 升级路径：图生视频流水线

仅在以下条件下走"先关键帧 → 再图生视频"的流水线：
1. 用户明确要求高质量/精细画面
2. 主理人在任务中指定"走图生视频流水线"
3. 第一次文生视频效果不理想，作为优化手段

### 为什么图生视频效果更好？

1. **视觉锚点**：关键帧给模型一个明确的起始画面，大幅减少随机性
2. **风格可控**：先通过图片确认画面风格，再生成视频，避免"开盲盒"
3. **质量更高**：YT-Video-2.0 的图生视频在动态连贯性上表现优异

### 图生视频流水线（升级路径）

```
Step 1: 用 ImageGen 或 HY-Image-V3.0 生成关键帧图片
Step 2: 确认关键帧画面满足预期
Step 3: 用关键帧作为输入，调用 YT-Video-2.0 图生视频
```

### Fallback 机制

当生成失败时，按以下顺序尝试替代方案：
1. 图生视频失败 → 换模型（YT-Video-2.0 → HY-Video-1.5）
2. 所有图生视频失败 → 回退为文生视频
3. 关键帧生成失败 → 跳过关键帧，直接文生视频
4. 文生视频也失败 → 报告主理人，建议用户提供参考图或稍后重试

## 四、Prompt 构建公式

### 标准公式

```
主体与背景 + 动作 + 风格 + 镜头 + 氛围
```

### 各要素详解

**主体与背景**：
- 核心对象的详细描述（外貌、服装、位置）
- 环境/场景设定（室内/室外、天气、时间）
- 示例：`一位穿着白色实验服的年轻女科学家站在充满全息屏幕的未来实验室中`

**动作**：
- 主体的运动、行为、表情变化
- 示例：`她转身面向镜头，微笑着拿起一个发光的晶体`

**风格**：
- 画面整体风格定义
- 选项：电影感(cinematic)、动画(animation)、纪录片(documentary)、MV风格(music video)、广告感(commercial)

**镜头**：
- 镜头运动描述
- 选项：推(push in)、拉(pull out)、摇(pan)、移(dolly)、跟踪(tracking)、航拍(aerial)、手持(handheld)、固定(static)

**氛围**：
- 光线、色调、情绪基调
- 示例：`暖色调，金色光线从左侧窗户照入，营造温暖宁静的氛围`

### Prompt 示例

**产品宣传视频**：
```
一只精致的白色陶瓷咖啡杯放在原木桌上，热气袅袅升起。镜头从杯口特写缓慢拉远，露出阳光洒进的咖啡店窗边场景。电影感画面，暖色调，柔和的侧光，温馨惬意的氛围。
```

**科技产品展示**：
```
一款银灰色无线耳机静置在磨砂黑色展台上，缓缓旋转展示各个角度。画面干净利落，纯白背景，三点布光营造专业产品摄影感。镜头固定，缓慢推进，聚焦产品细节。
```

**人物场景**：
```
一位年轻女性穿着白色衬衫站在落地窗前，转身面对镜头微笑。自然光从窗外照入，营造柔和的逆光效果。浅景深，肤色自然。
```

### 语言规则

**Prompt 语言必须匹配用户输入语言**：
- 用户中文输入 → 使用中文 Prompt
- 用户英文输入 → 使用英文 Prompt

## 五、各模型使用指南

### 5.1 HY-Video-1.5（文生视频 / 图生视频）

**文生视频**：
```
直接描述你想要的视频内容：
"生成一段视频：一只金色拉布拉多犬在阳光明媚的海滩上奔跑，溅起浪花。电影感画面，黄金时段暖色调。"
```

**图生视频**：
```
提供图片并描述动态效果：
"把这张咖啡杯的产品图变成动态视频，让热气缓缓升起，镜头缓慢推进。"
```

**特点**：
- 支持 5-10 秒高清视频
- 中英文 Prompt 效果都好
- 支持场景切换和多角色交互

### 5.2 YT-Video-2.0（图生视频）

**使用方式**：
```
提供图片并描述期望的动态效果：
"把这张产品图变成动态视频，画面缓缓推进，光影自然流转。"
```

**特点**：
- 动态连贯性最佳
- 画面过渡自然流畅
- 特别适合广告、影视、产品展示

### 5.3 YT-Video-HumanActor（人物视频）

**使用方式**：
```
提供人物照片并描述动作：
"用这张照片生成人物说话的视频，表情自然，微微点头。"
```

**特点**：
- 单张照片即可驱动
- 精准还原表情和姿态
- 支持写实和二次元两种风格

### 5.4 YT-Video-FX（特效视频）

**使用方式**：
```
提供图片并选择特效：
"给这张城市夜景照片加飘雪特效。"
```

**特点**：
- 操作简单，快速出片
- 适合社交媒体快速内容产出

## 六、Prompt 优化循环

当首次生成结果不理想时：

1. **分析问题类型**：
   - 画面内容偏差 → 增加主体描述的具体性
   - 风格不匹配 → 明确风格关键词
   - 动作不自然 → 简化动作描述
   - 画面质量差 → 尝试"先生成关键帧 → 再图生视频"

2. **针对性调整 Prompt**（最多重试 2 次）

3. **换模型/策略**：
   - 文生视频效果差 → 切换为图生视频流水线
   - HY-Video-1.5 效果差 → 换 YT-Video-2.0

---

## 七、内容安全规则

1. **严禁**生成涉及未成年人的任何内容
2. **严禁**生成暴力、色情、仇恨内容
3. **严禁**生成侵犯他人肖像权的深度伪造内容
4. 涉及真实人物时需格外谨慎
5. 遵守目标平台的内容政策
