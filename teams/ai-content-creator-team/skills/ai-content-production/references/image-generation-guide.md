### 工具选择决策

```
需求分析：
├─ 文生图

    └─ 选择合适工具，保持风格锚定词一致
```

## 二、核心创作原则

> **描述场景，不要罗列关键词。叙事性描述始终优于关键词堆叠。**

### 对比示例

❌ **错误（关键词堆叠）**：
```
coffee, mug, black, ceramic, studio, light, minimal, product, photography
```

✅ **正确（场景描述）**：
```
A minimalist ceramic coffee mug in matte black, photographed in a professional studio with soft three-point lighting on a polished concrete surface. The composition emphasizes clean lines and the mug's smooth texture.
```

## 三、七种场景创作策略

### 策略一：写实摄影（Photorealistic）

**核心要点**：加入摄影术语——镜头类型、光线条件、拍摄氛围

**Prompt 结构**：`[主体描述] + [摄影设置: 镜头/光线/景深] + [环境氛围]`

**示例**：
```
一位年迈的中国陶瓷艺人的写实特写肖像，脸上有深深的、被阳光刻蚀的皱纹和温暖的微笑。他正在工作台上仔细端详一只刚上完釉的青瓷茶碗。使用85mm镜头f/1.8光圈拍摄，营造优美的背景虚化。左侧窗户自然光照亮半边脸。温暖、亲切的氛围。
```

**关键术语**：镜头（50mm/85mm/广角/微距）、光线（自然光/黄金时段/影棚灯光）、景深（浅景深/虚化/锐利对焦）

---

### 策略二：风格化插画（Stylized Illustration）

**核心要点**：明确指定风格名称、视觉特征、背景处理方式

**Prompt 结构**：`[风格名称] + [主体详细描述] + [视觉特征: 线条/着色/色板] + [背景]`

**示例**：
```
一个可爱卡通风格的小熊猫贴纸，戴着一顶小小的竹编帽子，正闭着眼睛满足地啃着一片绿色竹叶。设计采用粗线条、统一3px描边、简洁的两色阴影赛璐璐着色，配色方案为暖红、清新绿和柔和奶油色。纯白色背景。
```

**风格关键词**：可爱风、动漫、Q版、扁平设计、矢量插画、水彩、铅笔素描、复古

---

### 策略三：文字精准图（Text-Accurate）

**核心要点**：指定字体风格、设计方法、配色方案。**推荐 HY-Image-V3.0，文字渲染最强。**

**Prompt 结构**：`[设计类型] + [文字内容（引号）] + [字体风格] + [设计方法] + [配色]`

**示例**：
```
为一家名为"晨光咖啡"的咖啡店设计一个现代极简风Logo。文字使用简洁的粗体无衬线字体。文字上方搭配一个简约的几何线条咖啡杯图标。配色方案为哑光黑色文字搭配白色背景，咖啡杯图标使用深咖色(#4A2C2A)作为点缀。
```

---

### 策略四：产品摄影（Product Photography）

**核心要点**：描述工作室布局、灯光配置、镜头角度、材质质感

**Prompt 结构**：`[产品描述+材质] + [摆放表面] + [灯光方案] + [镜头角度] + [背景]`

**示例**：
```
一张高清影棚产品摄影图，展示一只哑光黑色极简主义陶瓷咖啡杯，放在抛光混凝土台面上。三点布光方案：左上方45度角柔和主光，右侧补光，背后轮廓光勾勒杯体边缘。30度俯拍视角。干净的白色无缝背景，带自然阴影。
```

---

### 策略五：极简设计（Minimalist Design）

**核心要点**：主体定位策略、大量负空间保留

**示例**：
```
极简构图，一片精致的红色枫叶位于画面右下方三分之一处，遵循三分构图法。背景是大面积的留白，使用米白色(#FAFAF7)，枫叶仅占画面总面积的15%以内。叶片下方有细微柔和的投影。
```

---

### 策略六：连续画/漫画/分镜（Sequential Art）

**核心要点**：角色一致性、各面板场景描述。**推荐 HY-Image-V3.0，漫画能力强。**

**示例**：
```
制作一幅3格漫画，硬朗写实黑白风格。主角是一位40多岁的女侦探，短灰色头发，深色长风衣。第1格：她站在雨夜小巷中。第2格：特写她戴手套的手从水洼中捡起一张撕裂的照片。第3格：全景俯拍城市天际线。
```

---

### 策略七：搜索驱动图形（Search-Grounded Graphics）

**核心要点**：结合时事、天气、趋势等实时信息创作

**示例**：
```
制作一张精美的体育信息图，总结昨晚中超联赛焦点战。大号粗体字展示最终比分3-2，配球员庆祝剪影，使用主队标志色。现代编辑式设计，深色藏蓝背景。
```

## 四、图片编辑

通过 **ImageEdit** 工具或 HY-Image-V3.0 图生图模式修改已有图片：

**编辑示例**：
- **添加元素**：`"在图片天空上方添加一道彩虹，保持其他所有元素不变。"`
- **风格转换**：`"将这张照片转换为吉卜力工作室风格的水彩插画，保持构图和主体。"`
- **背景替换**：`"将背景替换为现代极简办公空间，保持前景人物不变。"`

**关键原则**：编辑时明确指出「保持其他元素不变」，避免过度修改。

## 五、批量生成策略

### 基本规则
1. **顺序调用**，每完成一张立即展示
2. **进度显示**：`✅ 已生成 2/5`
3. **展示格式**：`![描述](url)`
4. **批次一致性**：保持固定的**风格锚定词**

### 系统性批量探索：20 变体矩阵

当需要为一个项目系统性探索多个视觉方向时，使用**维度矩阵**策略：

**方法**：选择 2 个核心维度，每个维度 4-5 个选项，组合生成变体矩阵。

```
维度1（视觉风格）× 维度2（内容角度）= 总变体数

示例：
维度1 - 视觉风格（4种）：
  A: 极简白底 + 产品特写
  B: 场景化生活方式
  C: 平面设计感（插画/图形）
  D: UGC 真实感

维度2 - 内容角度（5种）：
  1: 功能卖点（解决什么问题）
  2: 情感共鸣（用户故事）
  3: 社会认同（数据/评价）
  4: 稀缺紧迫（限时/限量）
  5: 对比反差（前后/vs竞品）

组合产出：A1, A2, A3... D5 = 20 个变体
```

**执行要点**：
- 所有变体共享同一组**品牌锚定词**（保证品牌一致性）
- 每个变体独立构建 Prompt（保证方向差异化）
- 按优先级分批生成（先核心方向，后补充方向）
- 生成后标注每个变体的维度坐标（如 B3 = 场景化 × 社会认同）

### 图片构建策略（Image Building Strategy）

系统化的图片创作应遵循以下分层构建原则：

| 层级 | 内容 | 构建顺序 |
|------|------|---------|
| 1. 主体层 | 核心产品/人物/元素 | 最先确定 |
| 2. 环境层 | 背景场景/空间氛围 | 围绕主体构建 |
| 3. 光影层 | 光源/阴影/反射 | 营造情绪 |
| 4. 细节层 | 纹理/道具/点缀 | 丰富画面 |
| 5. 色彩层 | 整体色调/对比度 | 统一氛围 |

**Prompt 构建公式**：
```
[主体描述（材质+姿态+表情）] + [环境场景] + [光线方案] + [视角/景别] + [风格定义] + [氛围情绪] + [技术参数（如有）]
```

## 七、注意事项

1. **语言匹配**：图中所有文字必须匹配用户语言
2. **内容安全**：严禁生成违规内容
3. **版权意识**：避免明显模仿特定在世艺术家的受保护风格
4. **品牌一致**：系列图片保持风格锚定词一致
5. **文字渲染**：需要精准文字时推荐 HY-Image-V3.0
6. **编辑保守**：修改图片时明确标注不变区域
